## Codebase Patterns
- Log files use JSON format (one JSON object per line) - parse with JSON.parse()
- Log directory: app.getPath('userData') + '/logs'
- Daily rotating log files: YYYY-MM-DD.log format

## 2026-03-07 - US-001

- Files changed: apps/client/src/main/ipc.ts
- **Learnings for future iterations:**
  - Winston logger writes JSON format logs - need to parse each line as JSON

  - Logs should be sorted by timestamp (newest first) for display
