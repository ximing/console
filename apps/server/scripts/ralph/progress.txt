## Codebase Patterns
- This project uses routing-controllers with @CurrentUser() decorator for authentication
- Services are decorated with @Service() from TypeDI for dependency injection
- DTOs are defined in packages/dto and must be rebuilt after changes
- Controllers auto-register via glob import in src/controllers/index.ts
- Use ResponseUtil.success() and ResponseUtil.error() for API responses
