## Codebase Patterns
- Tool registry pattern: Define tools with id, name, nameEn, description, keywords, and category
- Use AI with confidence scores (>= 0.5 threshold) for tool matching
- Controllers use ResponseUtil for consistent response format
- AI tools identified by category: 'ai' in tool-registry.ts
- Command format (/command): Input starting with / is matched case-insensitively against tool IDs, then exact keywords, then partial keywords
