# 博客标签功能设计

## 概述

为博客系统添加完整的标签管理功能，支持在侧边栏管理标签、在编辑器中选择/创建标签、以及按标签筛选博客。

## 需求

1. **侧边栏新增「标签」Tab** - 管理所有标签，支持创建/编辑/删除/筛选
2. **编辑器内标签管理** - 在编辑态和预览态都可以选择/快速创建标签
3. **预设+自定义颜色** - 8种预设颜色 + 完整调色盘

## 架构

### 组件结构

```
Sidebar (侧边栏)
├── SidebarTabs (目录 | 最近 | 标签)
├── DirectoryTree (目录 Tab 内容)
├── RecentBlogList (最近 Tab 内容)
└── TagPanel (新增：标签 Tab 内容)
    ├── TagSearchBar
    ├── TagList (标签列表，支持点击筛选)
    ├── TagItem (单个标签，含编辑/删除)
    └── CreateTagButton → TagModal (创建/编辑弹窗)

BlogEditorPage (编辑器页面)
└── BlogEditorContent
    └── TagSelector (标签选择器，行内浮层)
```

### 数据流

- `TagService` (单例) - 管理所有标签状态
- `BlogEditorService.selectedTagIds` - 当前博客选中的标签
- `BlogService.blogs` - 博客列表，支持按 tagId 筛选

## 功能详情

### 1. 侧边栏「标签」Tab

#### 布局
```
┌─────────────────────┐
│ 🔍 搜索标签          │
├─────────────────────┤
│ ○ 全部 (12)         │  ← 默认显示全部博客
├─────────────────────┤
│ ● 标签1 (3)         │  ← 颜色点 + 名称 + 博客数
│ ○ 标签2 (5)         │
│ ○ 标签3 (1)         │
├─────────────────────┤
│ [＋ 新建标签]        │
└─────────────────────┘
```

#### 交互
- **点击标签**：筛选该标签下的博客，高亮选中态
- **悬停标签**：显示编辑(✏️)和删除(🗑️)图标
- **搜索标签**：实时过滤标签列表
- **新建/编辑标签**：打开 TagModal 弹窗

### 2. 标签管理弹窗 (TagModal)

#### 布局
```
┌───────────────────────────────┐
│ 新建标签           [× 关闭]  │
├───────────────────────────────┤
│ 名称:                          │
│ [__________________________]  │
│                               │
│ 颜色:                          │
│ ● ● ● ● ● ● ● ●             │  ← 8种预设颜色
│                               │
│ [自定义颜色]  [颜色选择器]     │
│                               │
│         [取消]  [保存]         │
└───────────────────────────────┘
```

#### 预设颜色
1. `#22c55e` - 绿色
2. `#3b82f6` - 蓝色
3. `#ef4444` - 红色
4. `#f59e0b` - 橙色
5. `#8b5cf6` - 紫色
6. `#ec4899` - 粉色
7. `#6b7280` - 灰色
8. `#14b8a6` - 青色

#### 颜色选择器
- 使用原生 `<input type="color">` 或 react-colorful
- 显示当前选中颜色的十六进制值

### 3. 编辑器内标签选择器

#### 位置
在标题下方、元信息行（字数、创建时间等）的右侧

#### 布局
```
┌─────────────────────────────────────────────────────┐
│ [标题输入框]                                        │
├─────────────────────────────────────────────────────┤
│ 字数: 123  |  标签: [标签1×] [标签2×] [＋ 添加]    │
├─────────────────────────────────────────────────────┤
│ [编辑器内容]                                        │
└─────────────────────────────────────────────────────┘
```

#### 编辑态交互
- **已有标签**：显示为可点击的药丸形状，点击取消选择（带×图标）
- **「＋ 添加」按钮**：点击展开浮层
- **浮层内容**：现有标签列表（可多选）+ 快速新建输入框

#### 预览态交互
- **已选标签**：纯展示，不可点击
- **点击标签区域**：弹出浮层，可选择标签

#### 浮层设计
```
┌─────────────────────────────────────┐
│ 添加标签                         [×] │
├─────────────────────────────────────┤
│ [🔍 搜索标签...]                   │
├─────────────────────────────────────┤
│ ☑ 标签1                            │
│ ☐ 标签2                            │
│ ☐ 标签3                            │
├─────────────────────────────────────┤
│ 新建标签: [输入名称...] [创建]     │
└─────────────────────────────────────┘
```

### 4. 按标签筛选博客

#### 入口
1. 侧边栏「标签」Tab - 点击标签
2. URL 参数 - `?tagId=xxx`（可选，支持分享链接）

#### 行为
- 筛选时更新 `BlogService.loadBlogs({ tagId })`
- 清空筛选：点击「全部」或点击当前选中标签

## API 端点（已存在，需确认）

- `GET /api/tags` - 获取所有标签
- `POST /api/tags` - 创建标签
- `PUT /api/tags/:id` - 更新标签
- `DELETE /api/tags/:id` - 删除标签
- `GET /api/blogs?tagId=xxx` - 按标签获取博客（需确认后端支持）

## 实现顺序

1. **TagService 完善** - 添加按名称搜索标签方法
2. **侧边栏标签 Tab** - TagPanel + TagItem + TagModal
3. **编辑器内标签选择器** - TagSelector 浮层组件
4. **标签筛选功能** - blogs.tsx 筛选逻辑
5. **样式优化** - 按设计系统规范调整颜色/间距

## 注意事项

- 标签删除时检查是否被博客使用，若有博客使用则提示用户
- 标签名称最大长度 20 字符
- 颜色选择器使用 design system 的绿色作为主色调
